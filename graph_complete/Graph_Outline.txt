## **ðŸ“¦ Installation Instructions**

### **Step 1: Extract All Batches**
Extract all 7 ZIP files into the **same directory**. They will automatically merge into the complete `nihiltheism-3d-map/` folder structure.

### **Step 2: Install Dependencies**
```bash
cd nihiltheism-3d-map

# Install Node.js dependencies
npm install

# Install Python AI service dependencies
cd ai-service
pip install -r requirements.txt
cd ..
```

### **Step 3: Configure Environment**
```bash
cp .env.example .env
# Edit .env with your MongoDB URI and other settings
```

### **Step 4: Setup Database**
```bash
# Make sure MongoDB is running locally or update MONGODB_URI in .env
npm run setup
npm run seed
```

### **Step 5: Run the Application**
```bash
npm run dev
```

This starts all 3 services simultaneously:
- **Frontend**: http://localhost:3000
- **Backend API**: http://localhost:8000  
- **AI Service**: http://localhost:8001
